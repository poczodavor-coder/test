# 🐍 贪吃蛇游戏 (Snake Game)

一款使用 **Python** 和 **Pygame** 构建的经典贪吃蛇游戏。

## 🎮 游戏玩法

- 控制蛇在屏幕中移动，吃掉随机出现的 **红色食物** 来增长身体并获得分数。
- 每吃掉一个食物，蛇的长度增加 1，得分 +1。
- 每得 **5 分**，屏幕会产生一次白色闪烁效果作为里程碑提示。
- **游戏结束条件：**
  - 🧱 撞到屏幕边界
  - 🔄 撞到自己的身体

## 🕹️ 控制方式

| 按键 | 功能 |
| --- | --- |
| `↑` | 向上移动 |
| `↓` | 向下移动 |
| `←` | 向左移动 |
| `→` | 向右移动 |
| `P` | 暂停 / 继续游戏 |
| `Q` | 游戏结束后退出 |
| `C` | 游戏结束后重新开始 |

> **注意：** 蛇不能直接反向移动（例如正在向右移动时不能直接向左）。

## ✨ 功能特性

- 🎯 **实时计分系统** — 左上角显示当前得分
- ⏸️ **暂停功能** — 按 `P` 键随时暂停/恢复
- 🚫 **防反向移动** — 防止蛇直接掉头导致意外死亡
- 💥 **里程碑闪烁** — 每 5 分触发一次视觉反馈
- 🌐 **中英文双语支持** — 自动检测系统字体，支持中文显示
- 🔄 **一键重开** — 游戏结束后按 `C` 即可重新开始

## 🛠️ 运行环境

- Python 3.x
- Pygame

## 🚀 快速开始

1. 安装依赖：

   ```bash
   pip install pygame
   ```

2. 运行游戏：

   ```bash
   python snake.py
   ```

## 📐 游戏参数

| 参数 | 值 |
| --- | --- |
| 窗口大小 | 600 × 400 |
| 蛇身方块大小 | 15 px |
| 游戏速度 | 12 FPS |
